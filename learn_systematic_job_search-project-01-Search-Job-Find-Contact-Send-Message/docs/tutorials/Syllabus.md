# Syllabus

This course teaches you how to use AI to systematically search for jobs, find insider contacts, and craft personalized outreach messages. By the end, you'll have a repeatable workflow for proactive job hunting that dramatically increases your response rates.

## 01-Search-Job-Find-Contact-Send-Message

Learn to use three AI-powered tools for systematic job hunting: (1) LinkedIn Job Search to find and filter matching positions using Claude for Chrome, (2) Contact Finder to identify alumni, peers, recruiters, and hiring managers at target companies, and (3) Cold Message Writer to generate personalized outreach. Includes a complete Nasdaq Data Scientist example with real outputs.
