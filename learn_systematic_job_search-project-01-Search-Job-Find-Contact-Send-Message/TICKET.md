# AI-Powered Systematic Job Search

## Objective

Learn how to use AI to automate three critical job hunting tasks: finding matching positions, identifying contacts at target companies, and writing personalized outreach messages. This is a follow-along tutorial—no graded deliverables, just hands-on practice.

Read the tutorial: [AI-Powered Systematic Job Search](https://github.com/easyscale-academy/learn_systematic_job_search-project/tree/01-Search-Job-Find-Contact-Send-Message/)

## Actionable Items

1. **Set up your profile materials** - Create your `resume.md`, `objective.md`, and other files in the `profile/` directory
2. **Run Tool 1: Job Search** - Generate the prompt, use Claude for Chrome to search LinkedIn, review the TSV output
3. **Run Tool 2: Contact Finder** - Pick a target job, generate the prompt, use Claude for Chrome to find contacts
4. **Run Tool 3: Cold Message** - Pick a contact, generate the prompt, use any AI to craft your message
5. **Send your first outreach message** - Copy the generated message and send it on LinkedIn

**Estimated time:** 60-90 minutes

## Checklist

- [ ] **Profile files created** - `resume.md` and `objective.md` are in the `profile/` directory
- [ ] **Job search completed** - Generated a TSV file with matching positions
- [ ] **Contacts identified** - Found at least 3-5 contacts for your target job
- [ ] **Message drafted** - Generated at least one personalized cold message
- [ ] **Message sent** - Sent at least one outreach message on LinkedIn

## Submission & Verification

This is a self-paced follow-along tutorial. There's no formal submission required. When you're done, you should have:

- A populated `profile/` directory with your materials
- Generated prompts in the `docs/` directory
- At least one outreach message sent on LinkedIn

Track your job applications in your own spreadsheet and follow up as needed.

## Grading Rubric

> **For instructors and /teach-check assistant** — Students may skip this section.

- **Profile completeness:** Check if `profile/resume.md` and `profile/objective.md` exist and contain substantive content
- **Prompt generation:** Verify that running `mise run gen-job-search` successfully generates `docs/linkedin-job-search-prompt.md`
- **Understanding of workflow:** Student can explain the three-tool pipeline and when to use Claude for Chrome vs any AI
- **Practical application:** Student has identified real contacts and drafted at least one message (even if not sent)
