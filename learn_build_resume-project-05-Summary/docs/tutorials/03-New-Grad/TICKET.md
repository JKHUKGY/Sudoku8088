# New Grad Resume Writing

## Objective

Learn how to write a compelling resume as a new grad without full-time work experience. Understand resume structure, when to use a Summary, how to write impactful project descriptions, and common mistakes to avoid.

Read the tutorial: [Build Your Resume with AI - Lesson 03](https://github.com/easyscale-academy/learn_build_resume-project/tree/03-New-Grad/)

## Actionable Items

1. Read the full tutorial from start to finish
2. Understand the recommended resume structure for new grads (Contact → Summary → Education → Skills → Projects → Work Experience)
3. Learn when and how to write an effective Summary for different job directions
4. Master the Action + Technical Detail + Impact formula for project descriptions
5. Study the common mistakes section and identify any you might be making
6. Understand strategy differences between campus recruiting and direct applications

**Estimated time:** 25-30 minutes

## Checklist

- [ ] **Understand resume structure** - Can explain the recommended section order for new grad resumes
- [ ] **Summary positioning** - Know when to write a Summary and that it goes above Education
- [ ] **Multiple resumes concept** - Understand why different job directions need different resume versions
- [ ] **Project description formula** - Can apply Action + Technical Detail + Impact to describe projects
- [ ] **Impact verification principle** - Understand "if you can't withstand follow-up questions, don't write it"
- [ ] **Common mistakes awareness** - Can identify the 8 common mistakes (tech overload, homework as projects, big words, etc.)

## Submission & Verification

This is a reading-only lesson with no code or deliverables. When you feel confident about the concepts, proceed to the next lesson.

## Grading Rubric

- **Structure understanding:** Student can explain why Summary goes at the top and why Projects are the most important section for new grads
- **Tailoring concept:** Student understands why one resume for all applications is ineffective
- **Project writing:** Student can differentiate between good and bad project descriptions using the formula
- **Impact verification:** Student can explain the principle of writing only verifiable impact numbers
- **Mistake identification:** Student can identify common resume mistakes and explain why they're problematic
